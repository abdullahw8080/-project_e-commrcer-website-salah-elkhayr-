Salla Al-Khair is a comprehensive online grocery store featuring an intuitive interface for easy browsing by category and seamless shopping. It offers flexible payment options, including cash on delivery and electronic cards. Once an order is placed, it is routed to the store manager for processing and fulfillment. Additionally, the system generates detailed reports on inventory, top-selling products, customer data, and high-demand areas to optimize performance and support effective decision-making.
سلة الخير هو متجر إلكتروني متكامل ومتخصص في بيع المواد الغذائية، يوفّر واجهة استخدام سهلة وبسيطة تُمكّن المستخدم من التسوق وتصفح المنتجات حسب الأقسام وإضافة المنتجات إلى سلة التسوق بكل سهولة. يتيح المتجر خيارات دفع متعددة تشمل الدفع النقدي (كاش) أو عبر البطاقة الإلكترونية. بعد إتمام الطلب، يتم إرساله مباشرة إلى مدير المتجر الذي يقوم باستلامه ومعالجته وإرساله للتنفيذ. كما يوفر النظام تقارير تفصيلية لمدير المتجر تشمل إدارة المخزون، والمنتجات الأكثر مبيعًا، وبيانات العملاء، والمناطق الأكثر شراءً، مما يساعد على تحسين الأداء واتخاذ قرارات فعّالة.
