# 🛒 متجر سلة الخير الإلكتروني (Salla Al-Khair E-Commerce)

موقع تجارة إلكترونية متكامل وسلس مصمم لتمكين المستخدمين من تسوق المواد الغذائية، الخضروات، الفواكه، ومستلزمات المنزل والأطفال عبر الإنترنت بسهولة ويسر، مع لوحة تحكم إدارية شاملة ونظام تقارير ذكي ومتقدم لمتابعة حركة المبيعات والمخزون.

## 🔍 المشكلة التي يحلها المشروع

يعاني الكثير من الأفراد في حياتهم اليومية من صعوبات أثناء شراء الاحتياجات المنزلية والغذائية الروتينية. يساهم متجر **سلة الخير** في حل المشكلات التالية:
- **توفير الوقت والجهد:** الاستغناء عن الذهاب الفعلي للسوبرماركت والوقوف في طوابير الانتظار الطويلة وحمل الأمتعة الثقيلة.
- **تسهيل اتخاذ القرار للعميل:** توضيح أسعار المنتجات بشكل مباشر قبل الشراء، وتبيين الخصومات والعروض الفعالة دون عناء البحث.
- **إدارة المخزون الذكية:** مساعدة أصحاب المتاجر في معرفة كميات السلع المتوفرة، والحد من الخسائر الناتجة عن انتهاء صلاحية المنتجات أو تراكم المنتجات الراكدة.
- **الافتقار إلى الإحصاءات والتقارير:** عدم معرفة أصحاب المشاريع التقليدية بالسلع الأكثر مبيعاً لديهم، أو المناطق الجغرافية الأكثر شراءً لتركيز الجهود التسويقية والخدمية عليها.

## 🌟 الفوائد والمزايا

### 👩‍💻 للعميل (المشتري):
- **تجربة مستخدم سلسة وعصرية:** تصفح سريع للمنتجات مقسمة حسب الفئات مع ميزة البحث السريع.
- **عروض وتخفيضات تفاعلية:** إمكانية رؤية السعر القديم والجديد ونسبة الخصم بوضوح.
- **سلة تسوق مرنة:** إمكانية زيادة ونقصان كمية المنتجات من واجهة العرض مباشرة أو داخل السلة مع حساب فوري للمجموع الإجمالي.
- **قائمة المفضلة:** لحفظ المنتجات التي يفضلها العميل للعودة إليها وطلبها لاحقاً.
- **تأكيد الطلبات التفصيلي:** كتابة تفاصيل التوصيل الجغرافية بدقة (المحافظة، المدينة، الحي، الشارع) والدفع الآمن عند الاستلام (كاش).

### 👨‍💼 للإدارة (المدراء والموظفون):
- **لوحة تحكم مركزية (Admin Dashboard):** واجهة واحدة تمنح المشرفين رؤية شاملة للمتجر وعدد الطلبات الواردة ورسائل العملاء الجديدة.
- **إدارة ديناميكية للمحتوى:** إضافة وتعديل وحذف المنتجات والأقسام والتحكم بالصور والأسعار والمخزون والوحدات.
- **نظام معالجة الطلبات:** قبول وتأكيد الطلبات المعلقة لشحنها، أو رفضها وإلغائها مع تتبع تفاصيل عنوان العميل الجغرافي ورقم هاتفه.
- **توزيع الصلاحيات:** إمكانية إنشاء حسابات جديدة للموظفين (مشرفين / إداريين) لإدارة العمليات اليومية.

## 🗄️ الهيكل البرمجي وقاعدة البيانات

تم بناء قاعدة بيانات المشروع `project_ecommerce` بشكل علاقي متين لضمان سرعة الاستعلامات وتكامل البيانات. يحتوي النظام على الجداول التالية:

| اسم الجدول | الوظيفة |
| :--- | :--- |
| `users` | يحتوي على بيانات العملاء والمشرفين (الاسم، البريد الإلكتروني، الهاتف، كلمة المرور المشفرة بـ MD5، وصلاحية الحساب). |
| `type_user` | يحدد نوع الحساب (مسؤول Admin / عميل User). |
| `products` | يخزن بيانات المنتجات (الاسم، السعر القديم والجديد، التخفيض، الكمية المتاحة، الوصف، الوزن، القسم). |
| `product_images` | يربط صور المنتجات بالمنتج المعني (يدعم رفع صور متعددة). |
| `product_unit` | يحدد وحدات بيع المنتجات (كيلو، حبة، كرتون، كيس، علبة، حزمة، الخ). |
| `section` | يخزن أقسام المتجر الرئيسية (خضروات، فواكه، مواد غذائية، بهارات، قسم الجملة، الخ) وصورها. |
| `pending_orders` | يخزن طلبات الشراء المؤقتة التي تنتظر مراجعة وقبول الإدارة. |
| `pending_orders_address` | يحتفظ بعناوين التوصيل المرتبطة بالطلبات المعلقة. |
| `pending_order_items` | يحتوي على تفاصيل المنتجات والكميات المحددة للطلبات المعلقة. |
| `orders` | يخزن الطلبات التي تم قبولها وتأكيدها من قبل الإدارة. |
| `orders_address` | عناوين توصيل الطلبات المؤكدة. |
| `order_items` | تفاصيل منتجات الطلبات المؤكدة. |
| `prefer_products` | يحتوي على المنتجات المفضلة للمستخدمين (قائمة الأمنيات). |
| `user_comments` | يستقبل تعليقات وآراء ومقترحات العملاء المعروضة والموجهة للإدارة. |
| `sessions` | مخصص لإدارة جلسات المستخدمين وحفظ بيانات السلة والاتصال عبر قاعدة البيانات لضمان بقاء الجلسة آمنة ومستقرة. |

## 🛠️ وظائف وميزات النظام

### 1. واجهة العميل (Storefront)
* **القائمة الرئيسية والأقسام:** تصفح المنتجات عبر الأقسام المختلفة مثل:
  - 🍎 الفواكه
  - 🥬 الخضروات
  - 🥫 المواد الغذائية
  - 👶 مستلزمات الأطفال
  - 🥤 المياه والمشروبات
  - 🧼 المواد المنزلية والتنظيف
  - 🌶️ البهارات
  - 📦 قسم الجملة
* **البحث المتقدم:** شريط بحث ذكي يبحث في عناوين وأوصاف المنتجات للوصول السريع.
* **التحكم بالكميات:** أزرار تفاعلية (`+` و `-`) تظهر عند الضغط على المنتج لتعديل الكمية المطلوبة بشكل مباشر دون الحاجة للانتقال لصفحة أخرى.
* **سلة تسوق تفاعلية:** تعرض المنتجات المضافة وسعر كل منها والمجموع النهائي مع إمكانية تحديث الكمية أو حذف المنتج فورياً.
* **نظام الحسابات والأمان:** تسجيل حساب جديد، تسجيل الدخول الآمن، والتحقق من صلاحية الجلسة لمنع الوصول غير المصرح به للملفات.

### 2. لوحة التحكم للمسؤولين (Admin Panel)
* **إحصائيات فورية:** إشعار الإدارة فوراً بعدد الطلبات المعلقة غير المعالجة والرسائل الجديدة الواردة من العملاء.
* **إدارة المنتجات والأقسام:**
  * إضافة منتج جديد: إدخال الاسم، السعر الأصلي، السعر بعد الخصم، الكمية بالمخزن، القسم، صورة المنتج، والوزن بالجرام، ووحدة البيع.
  * تعديل وحذف المنتجات الحالية وتحديث مستويات المخزون.
  * إضافة أقسام جديدة وتحديث صورها.
* **إدارة الطلبات والعملاء:** استعراض تفاصيل طلب المشتري (الاسم، الهاتف، العنوان بالتفصيل، المنتجات المطلوبة وكمياتها، والسعر الإجمالي) واتخاذ إجراء القبول (Confirm) أو الرفض (Reject).

### 3. محرك التقارير الذكي والمتقدم (Advanced Reporting Engine)
تتميز لوحة التحكم بنظام تقارير إدارية متكامل يساعد على اتخاذ القرارات التجارية الصائبة:
* **تقرير المنتجات الأكثر مبيعاً:** يوضح السلع الأكثر طلباً وربحية لزيادة المعروض منها.
* **تقرير المبيعات اليومية:** يعرض كشفاً تفصيلياً بالمبيعات اليومية ومقارنتها بالتواريخ السابقة وتتبع نمو المتجر.
* **تقرير المخزون والمنتجات:** يوفر إشرافاً دقيقاً على حالة المستودع والكميات المتبقية للسلع لتجنب نفاد المخزون.
* **تقرير الطلبات المعلقة:** تتبع مستمر للطلبات التي لم تُشحن بعد لضمان سرعة التوصيل.
* **تقرير المنتجات الراكدة (Slow Moving Items):** جرد بالمنتجات ذات الإقبال الضعيف لعمل عروض ترويجية وتخفيضات للتخلص منها قبل انتهاء صلاحيتها.
* **تقرير المناطق الأكثر شراءً:** يحلل التوزيع الجغرافي للمشترين لتحديد الأحياء والمناطق الأكثر ربحية وتوجيه سيارات التوصيل إليها.
* **تقرير العملاء الأكثر شراءً:** قائمة بالعملاء الأكثر تفاعلاً وولاءً للمتجر لتقديم مكافآت أو خصومات خاصة لهم.

## 💻 الأدوات والتقنيات المستخدمة

- **الواجهات الرسومية (Frontend):**
  - **HTML5 & CSS3:** لتصميم واجهات عصرية ومتجاوبة بالكامل مع مختلف الشاشات (الهواتف، الأجهزة اللوحية، الحواسيب).
  - **JavaScript (Vanilla JS):** لإضافة التفاعل مثل تعديل كميات السلال، إظهار وإخفاء القوائم الجانبية للهاتف، والبحث السريع.
  - **Swiper JS:** لتنفيذ سلايدر العروض الترويجية والمنتجات بشكل انسيابي وجذاب في الواجهة الرئيسية.
  - **Font Awesome & Boxicons:** لتوفير مكتبة غنية من الأيقونات الرمزية التوضيحية لسهولة التصفح.
  - **Google Fonts (Cairo):** استخدام خط "Cairo" العربي الأنيق المعتمد في أفضل التصاميم العربية الحديثة لضمان مقروئية ممتازة.

- **البرمجة الخلفية وقاعدة البيانات (Backend & Database):**
  - **PHP (8.x / 7.x):** لمعالجة الطلبات، وإدارة الجلسات بشكل مخصص، وحساب العمليات الرياضية للفواتير، والتواصل مع قاعدة البيانات.
  - **MySQL (MariaDB):** لإدارة وتخزين البيانات وحفظ الجلسات وتأمين العلاقات البرمجية بين الجداول.
  - **ترميز UTF-8 mb4:** لضمان عرض وقراءة الحروف العربية بشكل صحيح وخالٍ من الأخطاء في كل شاشات المتجر وقاعدة البيانات.
